Controlo Remoto E Soluções De Infecção
FINFLY WEB

Um dos maiores desafios na utilização de uma solução de controlo remoto é a instalação no sistema alvo, especialmente quando
apenas estão disponíveis poucas informações, como um endereço de e-mail, e não se consegue obter qualquer acesso físico.
O FinFly Web foi concebido para aplicar uma infecção remota e
de cobertura a um sistema alvo, utilizando uma vasta gama de
ataques baseados na web.
O FinFly Web proporciona uma interface do tipo apontar e
clicar, permitindo que o agente crie um código de infecção
personalizado simples, de acordo com os módulos seleccionados.

Exemplo 1 de utilização: Unidade de vigilância técnica

Exemplo 2 de utilização: Agência de serviços secretos

Depois de traçar o perfil de um alvo, a unidade criou um site
de interesse para o mesmo e enviou-lhe a ligação através
de uma quadro de discussão. Depois de abrir a ligação para
o site da unidade, uma solução de controlo remoto foi instalada
no sistema alvo e o alvo pode ser monitorizado a partir da
sede.

O cliente instalou o FinFly ISP no ISP principal do seu
país. Este foi combinado com o FinFly Web para, de forma remota, infectar os alvos que visitaram os sites de
ofensa governamental, injectando dissimuladamente o
código FinFly Web nos sites marcados.
